# angularjs-starter-kit
angularjs-starter-kit comes with:

* AngularJS 1.7.6
* Webpack 4
* Sass
* Karma
* ESLint
* angular-ui-router
* angular-resource

### start the development environment  
`npm start`  
  
### make production build
`npm run build`  
  
### fix ESLint warnings  
`npm run eslintfix`

### run unit tests
`npm run test`

files should be named as `*.test.js`
