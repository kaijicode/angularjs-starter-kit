# angularjs-starter-kit
angularjs-starter-kit comes with:

* AngularJS 1.7.6
* Webpack 4
* Sass
* Karma
* ESLint
* angular-ui-router
* angular-resource

### start the development environment  
`yarn start`  
  
### make production build
`yarn build`  
  
### fix ESLint warnings  
`yarn eslintfix`

### run unit tests
`yarn test`

files should be named as `*.test.js`
